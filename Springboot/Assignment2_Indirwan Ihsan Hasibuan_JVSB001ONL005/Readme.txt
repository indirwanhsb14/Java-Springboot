Latihan 1: Program untuk menentukan bilangan terbesar, terkecil dan rata-rata
    - Pada Latihan 1 terdapat kondisi harus menginput 3 variabel integer dimana terdapat X, Y dan Z.
    - Kemudian program memproses untuk menentukan nilai rata-tata dari 3 var tersebut dengan cara menjumlah kan
      ke-tiga var tsb kemudian dibagi oleh 3, hasil nya merupakan nilai rata-rata.
    - Menentukan bilangan terbesar dan terkecil bisa dibuat dengan menggunakan Syntax "Math.max" untuk menentukan
      bilangan terbesar, dan Syntax "Math.min" untuk menentukan bilangan terkecil.

Latihan 2: Program untuk menampilkan kelas dan posisi tempat duduk
    - latihan 2 diberi kondisi untuk menghasilkan input sesuai kondisi yang sudah ditentukan.
    - Terdapat 6 var yang harus di input dengan ditentukan kelas dari 0,0 s/d 1,2 atau saya sebut x,y.
    - Proses untuk menampilkan kelas adalah dengan menggunakan perulangan for dimana x dan y nilai awal nya 0.
    - Kemudian di cetak nilai x dan y tsb.
    - Untuk posisi dengan array dengan nilai yang sudah ditentukan.
    - Kemudian di cetak dengan posisi yang sudah ditentukan sesuai output latihan 2 yang sudah ditentukan.

Latihan 3: Program menentukan harga diskon dengan kondisi yang ditentukan
    - Minimum mendapatkan diskon total pembelian 1000000, diskon yg di dapatkan 10%.
    - Output nya harga setelah diskon
    - Untuk menentukan diskon nya dengan menggunakan if, jika memenuhi maka mendapatkan diskon, jika tidak akan
      masuk ke else.
    - Kemudian output nya adalah harga setelah diskon atau tidak mendapatkan diskon.

Latihan 4: Menetukan Tahun Kabisat
    - Input adalah memasukkan tahun yang ingin di cek apakah kabisat atau bukan.
    - Kemudian menggunakan percabangan kondisi if, else if dan else untuk proses perhitungan tahun nya.
    - Output nya adalah penentuan tahun kabisat atau tidak tahun kabisat.
